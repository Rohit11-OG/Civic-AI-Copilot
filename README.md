### ğŸ”§ **Installation Steps**

<details>
<summary><b>Step 1: Clone the Repository</b></summary>

```bash
git clone https://github.com/Rohit11-OG/Civic-AI-Copilot.git
cd Civic-AI-Copilot
```

</details>

<details>
<summary><b>Step 2: Backend Setup</b></summary>

```bash
# Navigate to backend folder
cd backend

# Create virtual environment
python -m venv venv

# Activate virtual environment
# On Linux/Mac:
source venv/bin/activate
# On Windows:
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Create environment file
echo "GROQ_API_KEY=your_api_key_here" > .env
# Replace 'your_api_key_here' with your actual Groq API key
```

</details>

<details>
<summary><b>Step 3: Start the Backend Server</b></summary>

```bash
# Make sure you're in backend/ folder with activated venv
python app.py

# Server will start at http://localhost:5000
# You should see: "Server running on port 5000"
```

</details>

<details>
<summary><b>Step 4: Launch the Frontend</b></summary>

**Option A: Direct File Opening**
```bash
# Open frontend/index.html directly in your browser
# File path: /path/to/civic-ai-copilot/frontend/index.html
```

**Option B: Local HTTP Server (Recommended)**
```bash
# Open new terminal, navigate to frontend folder
cd frontend

# Start simple HTTP server
python -m http.server 8080

# Open browser and visit: http://localhost:8080
```

</details>
